Links (Work in progress)

Links have a very simple structure where the rel and href field are required. The rel field is for locating the link in order to follow it. The href fields contains the mechanism for getting the related information. This mechanism could trigger a request to a backend or it could be an expression that must be executed in the client side upon the response.

Links to backend

 "#links": [
       {
        "href":"http://api.com/products/1/reviews",
        "rel":"reviews"
       }          
   ]
     
   GET http://api.com/products/1/reviews

Client side links

 "#links": [
       {
        "href":"sailpath://$.manufacturers.13324",
        "rel":"manufacturer"
       }          
   ],
 "manufacturers": {
     "13324": {
                  ....
       }
   }
