Meta

The meta section is used for adding any metada the API wants to publish. For now it's a bit experimental and Sail only defines the field headers which means any request that departs from the current response must have the specified headers.

 "#meta": [
       {
        "headers":[
               "X-TID":"3d3424ed"
         ]
       }          
   ]
     
   GET http://api.com/resources/1
   X-TID:3d3424ed
