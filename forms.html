Forms

Forms are the mechanism for decoupling actions like creation, modification or deletion. When using forms the client doesn't need to cope with URLs, HTTP verbs, headers and even in some cases with bodies. By default forms submmit a JSON body, but other body formats can be specified. All forms must be specified in a #form section.

  "#forms": [
        {
         "id":"add",
         "action":"http://api.com/cities",
         "method":"POST",
         "inputs": [
               {
                "name":"name",
                "value":
               },
               {
                "name":"coordinate.latitude",
                "value":
               },
               {
                "name":"coordinate.longitude",
                "value":
               }
            ]
        }          
     ]
